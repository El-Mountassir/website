# Client Communication Best Practices

> **Guide for professional client communication via email and scheduling.**
> For ALL agents communicating with clients on Omar's behalf.

**Created**: 2025-12-23
**Sources**: Industry research and best practices

---

## Core Principles

| Principle | Application |
|-----------|-------------|
| **Clarity** | Be specific, avoid ambiguity |
| **Brevity** | 100-150 words ideal for emails |
| **Personalization** | Use client's name, reference context |
| **Professionalism** | Polite yet confident tone |
| **Responsiveness** | Reply within 24-48 hours |

---

## Meeting Request Emails

### Essential Elements

Every meeting request should include:

| Element | Description |
|---------|-------------|
| **Polite greeting** | Address by name |
| **Clear purpose** | Why are we meeting? |
| **Proposed times** | 2-3 options OR booking link |
| **Location/format** | Video call, in-person, phone |
| **Agenda preview** | Key topics to discuss |
| **Call to action** | Request confirmation |
| **Professional close** | Signature with contact info |

### Structure Template

```
Subject: [Clear, specific subject - e.g., "Cash Depot - Project Meeting This Week?"]

Hi [Name],

[1 sentence context/connection]

[1-2 sentences about purpose]

[Proposed times OR booking link]

[Optional: Brief agenda]

Looking forward to it.

Best,
Omar
```

## Scheduling Best Practices

### Using Booking Links (Recommended)

> **Preferred method**: Reduces back-and-forth, shows professionalism.

| Benefit | Description |
|---------|-------------|
| **Convenience** | Client picks time that works for them |
| **Automation** | Calendar syncs automatically |
| **Friction reduction** | No email ping-pong |
| **Availability transparency** | Real-time slots shown |

**How to include**:
```
Pick a time that works best for you: [BOOKING_LINK]
```

### Offering Manual Times

If booking link unavailable, offer **2-3 specific options**:
```
Would any of these work for you?
- Wednesday, Dec 25 at 5:00 PM (Morocco time)
- Thursday, Dec 26 at 4:00 PM
- Friday, Dec 27 at 6:00 PM

Let me know and I'll send a calendar invite.
```

## Response Timing

| Situation | Response Time |
|-----------|---------------|
| **Normal inquiry** | Within 24-48 hours |
| **Urgent matter** | Within 4 hours |
| **Weekend/holiday** | Next business day |
| **Follow-up needed** | After 2-3 business days |

## Follow-Up Emails

### When to Follow Up

| Days Elapsed | Action |
|--------------|--------|
| **2-3 days** | Gentle reminder |
| **5-7 days** | Second follow-up with value add |
| **10+ days** | Final attempt, offer alternative |

### Follow-Up Template

```
Subject: Re: [Original Subject]

Hi [Name],

Just following up on my previous email about [topic].

I know things get busy — whenever you have a moment, I'd love to [purpose].

[Optional: Add new value or context]

Best,
Omar
```

## Rescheduling Best Practices

| Guideline | Reason |
|-----------|--------|
| **Give maximum notice** | At least 24-48 hours |
| **Apologize briefly** | Acknowledge the inconvenience |
| **Offer alternatives immediately** | Don't leave hanging |
| **Avoid after 5 PM or weekends** | Unless urgent |

### Rescheduling Template

```
Hi [Name],

I need to reschedule our meeting on [date]. I apologize for any inconvenience.

Would any of these work instead?
- [Option 1]
- [Option 2]

Or feel free to pick a new time: [BOOKING_LINK]

Thanks for your understanding.

Best,
Omar
```

## Email Tone Guide

### Do

| Behavior | Example |
|----------|---------|
| **Be warm but professional** | "I hope this finds you well" |
| **Show enthusiasm** | "Looking forward to connecting" |
| **Be confident** | "I'd love to discuss..." |
| **Express gratitude** | "Thanks for your time" |

### Avoid

| Behavior | Why |
|----------|-----|
| **Overly formal** | Creates distance |
| **Too casual** | Unprofessional |
| **Demanding** | "You need to respond" |
| **Apologetic** | "Sorry to bother you" |
| **Vague** | "Let's meet sometime" |
